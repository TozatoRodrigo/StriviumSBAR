## Short Description

Gestao segura de internacoes, pacientes e evolucoes clinicas para equipes de saude.

## Full Description

Strivium Link ajuda equipes de saude a organizar internacoes, pacientes, equipes medicas e evolucoes clinicas em um fluxo digital unificado.

O aplicativo foi desenhado para profissionais autorizados acompanharem tarefas clinicas, registrarem evolucoes e manterem informacoes operacionais acessiveis durante a rotina hospitalar.

Strivium Link e uma ferramenta de apoio operacional. O app nao substitui julgamento profissional, nao realiza diagnostico autonomo e nao deve ser usado como dispositivo medico regulado sem validacao e autorizacao apropriadas.

## Review Notes

Use the demo account provided in App Review / Play Console to access a workspace with fictitious patients, fictitious hospitalizations, and sample clinical evolutions.

For the first store release, experimental SBAR voice dictation and AI dictation are disabled. Any screens mentioning SBAR are intended for structured clinical documentation support and require professional review.
